# Smart-Parking-Management-System
## Overview
This project presents a Smart Parking Management System designed to tackle urban parking challenges using innovative technology. Developed as part of the coursework at the American University of Science & Technology, this system integrates Arduino and MATLAB to optimize parking space utilization and improve efficiency.

## Objectives
The main objectives of this project are:
* Design a low-cost and effective parking management solution using Arduino microcontroller technology.
* Integrate MATLAB to create a user-friendly Graphical User Interface (GUI) for system control and monitoring.
* Implement a prototype to demonstrate the feasibility and effectiveness of the smart parking system in a simulated environment.

## Tools Used
* Arduino Uno R3: Used as the microcontroller to interface with sensors and control components.
* MATLAB: Employed to develop the GUI and manage the system logic.
* IR Sensors, Servo Motor, LEDs: Hardware components utilized for vehicle detection, barrier control, and system status indication.

## Key Features
* Graphical User Interface (GUI): Provides intuitive control options and real-time feedback on parking space availability.
* Automated Parking Barrier: Controlled by Arduino based on inputs from IR sensors, ensuring efficient entry and exit management.
* LED Indicators: Visualize system status (ON/OFF) and parking availability at a glance.

## Conclusion
The Smart Parking Management System demonstrates how MATLAB and Arduino technology can be leveraged to address real-world challenges effectively. By providing a user-friendly interface and automating parking operations, the system aims to enhance convenience and optimize space utilization in urban environments.
